Bat-Flappy

Use the SPACEBAR or MOUSE CLICK to make it fly.

This is a flapper project for those interested in learning game development using Unity engine.
It features:

  • 2D Grids for scenary construction
  • Random scenary instantiation and destruction
  • Sprite atlas
  • Responsive GUI
  • Score keeping
  • Unity Ads
  • And many more features that you can check in the source code if you liked it

The source code for this project is freely availabe on GitHub under GPL-2.0 license.
